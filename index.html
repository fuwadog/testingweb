About Me

I am using the MERN Stack to develop myself as a full stack developer. My main goal for now is to develop applications that work independently of the tutorial environment. I also enjoy finding out how systems work from start to finish; this includes setting up databases, running APIs, optimizing front end performance and creating a positive user experience.

Through hand on development and testing, I have learned the most about how to develop, how to test, how to break, and how to fine-tune a system. This type of approach has shaped my method of designing systems and solving problems.

Over the last few weeks I have been researching AI, and specifically integrating AI into web applications through n8n automation and testing AI-driven features in real world projects. Currently, I find it very exciting to build applications that are useful and provide time and effort savings.
